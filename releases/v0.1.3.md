# V(0.1.3)

### New Features

#### Undo / Redo Support
- Added full **Undo (`Ctrl + Z`)** and **Redo (`Ctrl + Shift + Z`)** functionality.
- Enables safer experimentation and faster recovery from mistakes while editing files.

#### Drag and Drop File Support
- Files can now be dragged and dropped directly into the app.
- Streamlines opening and organizing files without relying on manual file selection.

#### Prolog Language Support
- Added syntax and tooling support for **Prolog**.
- Expands the range of supported languages for logic programming and academic use cases.

#### Git Integration
- Introduced an initial **Git interface**.
- Allows users to connect projects to a Git repository, laying the groundwork for version control workflows directly inside the app.

---

### Search & Navigation Improvements

#### Improved In-File Search
- Fixed issues with searching within files to ensure accurate and reliable results.
- Search now correctly identifies matches and navigates between them.

#### File Search Bar
- Added a dedicated **file search bar**.
- Makes it easier to quickly locate files in large projects without manually expanding directories.

---

### Power User Enhancements

#### Emacs Command Support
- Added support for **Emacs-style commands**.
- Enables experienced users to leverage familiar keybindings and editing patterns for faster navigation and editing.

---

### UX & Behavior Fixes

#### File Naming Capitalization
- Removed automatic capitalization when creating new files.
- File names now preserve the exact casing entered by the user, improving consistency and avoiding unexpected renames.

#### Context Menu Improvements
- Right-clicking a directory now provides options to:
  - **Add Folder**
  - **Add File**
- Makes project structure management faster and more intuitive.

#### Smart Folder Expansion
- When a file is created inside a **collapsed directory**:
  - The directory automatically opens.
  - The newly created file is highlighted.
- Reduces confusion and makes file creation feedback immediate and visible.

---

### Security & Stability

#### Security Review
- Conducted a review of potential security bugs.
- Addressed identified issues to improve overall stability and safety.
